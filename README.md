## plusコマンド
![test](https://github.com/basiliskv/robosys2023/actions/workflows/test.yml/badge.svg)
### 使い方
標準入力で数字を行ごとに読み取り、総和を出力します。  
**入力例**  

```
  $ seq 5 | plus
  15
  $ seq 9 | ./plus 
  45
```
